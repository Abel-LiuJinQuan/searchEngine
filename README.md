# searchEngine
基于ElasticSearch和Java实现的分布式搜索引擎

### 版本列表
#### v0.0.1

1.使用 ElasticSearch 搭建 Java 底层调用逻辑，封装 ES 的 API，
为后续搜索引擎平台开发提供简易API

2.编写接口，实现插入文本到ES，并能够实现搜索文本任意关键字查找文本信息

3.基于Vue.js编写简易界面，实现添加文本、搜索关键字功能
